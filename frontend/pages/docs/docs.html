التوثيق الرسمي لـ OnePass API

كل ما تحتاجه لدمج زر تسجيل الدخول وحماية البريد الوهمي للمستخدمين.

مقدمة

OnePass API تتيح لك دمج نظام تسجيل الدخول عبر البريد الوهمي في تطبيقاتك ومواقعك بسهولة. وفر تجربة مستخدم سلسة وآمنة مع حماية خصوصية مستخدميك.

المصادقة Endpoints أمثلة برمجية

المصادقة

للوصول إلى OnePass API، تحتاج إلى مصادقة باستخدام مفتاح API الخاص بك. يمكنك الحصول على المفتاح من صفحة مفاتيح API.

مثال JavaScript


// مصادقة باستخدام OnePass API
fetch("https://api.OnePass.me/auth", {
  method: "POST",
  headers: {
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_CLIENT_SECRET"
  },
  body: JSON.stringify({ code: "USER_AUTH_CODE" })
})
.then(res => res.json())
.then(data => console.log(data))
.catch(error => console.error("Error:", error));

Endpoints

Endpoint المصادقة

POST
https://api.OnePass.me/auth

Request Parameters

{
  "code": "string",     // كود المصادقة من المستخدم
  "client_id": "string" // معرف العميل (اختياري)
}

Response Example

{
  "success": true,
  "data": {
    "user_id": "user_123",
    "email": "user@OnePass.me",
    "name": "اسم المستخدم",
    "avatar": "https://example.com/avatar.png"
  }
}

Endpoint البريد الوهمي

GET
https://api.OnePass.me/user/email

Headers المطلوبة

Authorization: Bearer YOUR_ACCESS_TOKEN

أمثلة برمجية

Node.js مثال

Node.js 

const axios = require('axios');

async function authenticateUser(code) {
  try {
    const response = await axios.post('https://api.OnePass.me/auth', {
      code: code,
      client_id: process.env.CLIENT_ID
    }, {
      headers: {
        'Authorization': `Bearer ${process.env.CLIENT_SECRET}`,
        'Content-Type': 'application/json'
      }
    });
    return response.data;
  } catch (error) {
    console.error('Authentication failed:', error);
    throw error;
  }
}

Python مثال

Python 

import requests

def authenticate_user(code):
    headers = {
        'Authorization': f'Bearer {CLIENT_SECRET}',
        'Content-Type': 'application/json'
    }
    data = {
        'code': code,
        'client_id': CLIENT_ID
    }
    
    response = requests.post(
        'https://api.OnePass.me/auth',
        headers=headers,
        json=data
    )
    
    if response.status_code == 200:
        return response.json()
    else:
        raise Exception(f'Authentication failed: {response.text}')

الإرشادات الأمنية

لا تشارك Client Secret

احتفظ بـ Client Secret في بيئة آمنة على الخادم ولا تعرضه في الكود الأمامي.

خزّن التوكن في الخادم

تأكد من تخزين tokens الوصول في الخادم وليس في متصفح المستخدم.

استخدم HTTPS دائمًا

تأكد من استخدام HTTPS في جميع طلبات API لحماية البيانات.
